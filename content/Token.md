Token - the basic unit of information that the LLM operates with.

Unlike Shannon's information theory, where the atomic unit of information is a bit, the token has semantic content.

Mathematically, the token is a discrete symbol from the set $\Omega$, which is the space of elementary semantic units ([[Token Alphabet|token alphabet]]).
